Virtual fitting room integration that allows customers to try on clothes virtually before purchasing using AI-powered body detection technology.

== Description ==

Neuralyn TRYON Connector for WooCommerce is a virtual fitting room plugin that integrates the Neuralyn TRYON SaaS platform with your WooCommerce store. It allows your customers to virtually try on clothing items before making a purchase, increasing customer confidence and reducing return rates.

= Key Features =

* **Virtual Try-On**: Customers can see how clothing items look on them using AI-powered body detection
* **Multiple Button Placements**: Configure where the Try-On button appears on product pages
* **Customer Classification**: Automatically classify customers as guests, registered users, or buyers
* **Product Tabs Integration**: Add Try-On functionality as a product tab
* **Shop Listing Support**: Show Try-On buttons on shop and category pages
* **Multi-language Support**: English, Portuguese (Brazil), and Spanish translations included

= Requirements =

* WordPress 6.0 or higher
* WooCommerce 7.0 or higher
* PHP 7.4 or higher
* Active Neuralyn TRYON subscription
* HTTPS enabled

= How It Works =

1. Install and activate the plugin
2. Enter your Neuralyn TRYON license key in the settings
3. Configure button placement and style
4. Customers can now virtually try on your products

== Installation ==

1. Upload the `neuralyn-tryon` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce > Settings > Neuralyn TRYON to configure the plugin
4. Enter your license key from your Neuralyn Dashboard

== Frequently Asked Questions ==

= Where can I get a license key? =

You can obtain a license key from your Neuralyn Dashboard at https://neuralyn.ai/dashboard

= Which pages show the Try-On button? =

By default, the button appears on product pages. You can also enable it for shop/category listings. The button is automatically hidden on cart, checkout, and order confirmation pages.

= Can I customize the button appearance? =

Yes! Go to your Neuralyn dashboard at https://neuralyn.ai/dashboard and change the button style under your subscription, e-commerce option.

= Is the plugin compatible with HPOS? =

Yes, the plugin is fully compatible with WooCommerce High-Performance Order Storage (HPOS).
